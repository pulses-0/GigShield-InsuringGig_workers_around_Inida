# 13. Technology Stack

The GigShield platform uses a modern, scalable technology stack optimized for real-time processing, data-driven decision making, and rapid development.

---

## 13.1 Overview

The system is built using a layered stack:

- frontend (client application)  
- backend services  
- data storage  
- AI/ML components  
- integrations  

---

## 13.2 Frontend

- React / Next.js  
- Progressive Web App (PWA)

### Responsibilities

- user interface and interaction  
- dashboards and visualization  
- real-time alerts  

---

## 13.3 Backend

- Node.js / FastAPI  

### Responsibilities

- API handling  
- business logic execution  
- service orchestration  

---

## 13.4 Database

- PostgreSQL  

### Usage

- structured data storage  
- transactions and audit logs  

---

## 13.5 AI/ML Layer

- Python  
- scikit-learn  

### Models

- Random Forest  
- Gradient Boosting  
- Isolation Forest  

---

## 13.6 Payments

- Razorpay (test mode)  
- UPI simulation  

---

## 13.7 Notifications

- Firebase / Web Push  

---

## 13.8 Key Characteristics

- modular and scalable  
- easy to deploy  
- cost-efficient  
- production-ready  

---
