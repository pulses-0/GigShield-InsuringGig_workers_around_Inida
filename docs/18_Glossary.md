# 18. Glossary

This section defines key terms used in the system.

---

## Terms

GigShield Platform  
The overall system providing parametric insurance services.

Worker Portal  
Interface used by delivery partners.

Admin Portal  
Interface used by administrators.

Risk Engine  
Module for calculating risk scores and premiums.

Trigger System  
Component that detects disruption events.

Claims Engine  
Module that creates and processes claims.

Fraud Detection  
System that validates claim authenticity.

Payout Service  
Handles payout processing.

Coverage Period  
Weekly insurance duration.

Disruption Event  
External event affecting earnings.

Delivery Zone  
Geographic area where the user operates.
